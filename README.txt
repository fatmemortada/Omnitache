OMNITACHE WEBSITE - LOCAL PREVIEW

How to open the website on your laptop:
1. Unzip the folder.
2. Open the folder named omnitache-website.
3. Double-click index.html.
4. The website will open in your browser.

To edit the email used by the meeting form:
1. Open script.js.
2. Find this line:
   const recipient = 'info@omnitache.com';
3. Replace it with the real email address.
4. Save the file.

Files included:
- index.html
- styles.css
- script.js
- README.txt
